### Special Characters

#### Virama (හල් කිරීම)
| Unicode | Character | Name | Usage |
|---------|-----------|------|-------|
| U+0DCA  | ්         | al-lakuna/virama | Removes inherent vowel: ක + ් = ක් |

#### Zero-Width Joiner (ZWJ) - CRITICAL!
| Unicode | Character | Name | Usage |
|---------|-----------|------|-------|
| U+200D  | (invisible) | ZWJ | Forms conjuncts: ක + ් + ZWJ + ය = ක්‍ය |

**IMPORTANT:** ZWJ (U+200D) is ESSENTIAL for proper Sinhala rendering!
- WITH ZWJ: භාග්‍යවතුන් (correct)
- WITHOUT ZWJ: භාග්යවතුන් (broken)

### Common Conjuncts (යුක්තාක්ෂර)

These MUST use ZWJ (U+200D):
- ක්‍ය (kya) = ක + ් + ZWJ + ය
- ක්‍ර (kra) = ක + ් + ZWJ + ර
- ක්‍ව (kva) = ක + ් + ZWJ + ව
- ග්‍ර (gra) = ග + ් + ZWJ + ර
- ප්‍ර (pra) = ප + ් + ZWJ + ර
- ත්‍ර (tra) = ත + ් + ZWJ + ර
- ශ්‍ර (śra) = ශ + ් + ZWJ + ර

## Validation Rules

### ✅ Valid Sinhala Text Must:
1. Use ONLY characters from U+0D80-U+0DFF range
2. Include ZWJ (U+200D) for proper conjuncts
3. Have proper vowel sign placement
4. Use virama (්) correctly before consonants
5. Not have double virama (්්)
6. Not have multiple consecutive ZWJ

### ❌ Invalid Patterns:
- Vowel signs before virama: කැ් (wrong)
- Virama without ZWJ before ya/ra/va: ක්ය (broken, should be ක්‍ය)
- Double virama: ක්් (typing error)
- Foreign script characters (Tamil, Hindi, Thai, etc.)

## Common Mistakes

### 1. Missing ZWJ
❌ Wrong: භාග්යවතුන් (broken rendering)
✅ Correct: භාග්‍යවතුන් (proper rendering)

### 2. Foreign Characters
❌ Wrong: භාග்யవතුන් (contains Tamil ்)
✅ Correct: භාග්‍යවතුන් (pure Sinhala)

### 3. Improper Vowel Placement
❌ Wrong: කැ් (vowel before virama)
✅ Correct: ක්ැ (virama before vowel, though rare)

## Testing Your Text

Use this regex to validate pure Sinhala:
```regex
^[\u0D80-\u0DFF\u200D\s.,;:!?()\[\]{}'"0-9]+$
```

This allows:
- All Sinhala characters (U+0D80-U+0DFF)
- ZWJ (U+200D)
- Whitespace
- Common punctuation
- Numbers
